给定一个链表，判断链表中是否有环。

如果链表中有某个节点，可以通过连续跟踪 `next` 指针再次到达，则链表中存在环。 为了表示给定链表中的环，我们使用整数 `pos` 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 `pos` 是 `-1`，则在该链表中没有环。**注意：`pos` 不作为参数进行传递**，仅仅是为了标识链表的实际情况。

如果链表中存在环，则返回 `true` 。 否则，返回 `false` 。

 

**进阶：**

你能用 *O(1)*（即，常量）内存解决此问题吗？

## 题解

这道题是双指针-快慢指针的典型应用。利用一个慢指针，每次只走一步，再利用一个快指针，每次走两步，如果链表中存在环，那么迟早快指针会追到慢指针；如果不存在环，快指针会达到链表的末尾。

代码如下：

```c++
// @lc code=start
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    bool hasCycle(ListNode *head) {
        if(head == nullptr || head->next == nullptr)
            return false;

        ListNode* slow = head;
        ListNode* fast = head->next->next;
        if(fast == nullptr)
            return false;
        
        while(fast != slow){
            slow = slow->next;
            fast = fast->next;
            if(fast == nullptr)
                return false;
            fast = fast->next;
            if(fast == nullptr)
                return false;
        }
        return true;
    }
};
// @lc code=end

```
